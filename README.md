# CogniFi — Cognitive Wi-Fi Security

> *The Future of Smart Cyber-Solutions through Intelligent Behavioral Analysis of Public Networks*

---

## About The Project

Public Wi-Fi is everywhere—cafés, airports, hotels, and college campuses. But how many of us actually check if the network is safe before connecting? 

**CogniFi** is our attempt to solve this problem. It's a client-side behavioral trust scoring system that analyzes Wi-Fi metadata—like DNS response times, latency patterns, and beacon intervals—to detect threats like evil twin attacks and DNS spoofing. Before you connect to a public network, CogniFi gives you a simple safety score:

✅ Safe (70-100)  
⚠️ Suspicious (30-70)  
🔴 Risky (0-30)

No packet inspection. No infrastructure changes. Just a lightweight tool that empowers everyday users to make informed decisions.

This project is our **Bachelor of Engineering (CSE)** final year project at Canara Engineering College, Mangaluru, under the guidance of Mr. Alok Ranjan.

## Objectives

1. **Study** behavioral patterns of public Wi-Fi networks by capturing metadata including DNS responses, latency variations, and gateway consistency.

2. **Detect and classify** suspicious activities such as evil twin attacks and DNS spoofing using machine learning.

3. **Develop and evaluate** a real-time trust scoring system that translates technical anomalies into simple safety indicators for users.
